Administrative Functionality

The Admin section of the e-MissionManager application allows the user to configure any administrative functionality within the application. This includes adding further users to the application, configuring meters, configuration of conversion factors and the activity log.

Managing Users

The first option in the admin section is "Manage Users" and can be found by clicking Admin > Manage Users. This page lists all known users in of the application within your organisation. The users will be displayed in a table as shown below in Fig.41

Fig.41 Manage Users

The table above lists both the users and has 3 different buttons for each user.

  • Show: This button shows all the details of this user.
  • Edit: This button allows you to alter the details of the specified user.
  • Delete: This button allows you to delete a particular user so they won't have rights access the application.
