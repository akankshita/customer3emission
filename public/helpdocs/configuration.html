Configuration of Meters
The first thing necessary to do will be to configure your organisational structure. In the main menu, click Admin > Configure Meters. Fig.1 below is an example of an organisation structure that has already been setup.
Fig.1 Configure Sources
This page requires you to load the structure of your organisation so the e-MissionManager application will be able to display the data in terms of your organisation. Entering information that doesn't resemble that of your organisation will result in incorrect data being shown.*
The first thing you will do is enter each premises into the system that you are monitoring emissions for. Once you have entered a new premises in the "New Premises Details" dialog box, it will then appear in the tree diagram on the left.
Once this is done you must click on the premises in the tree diagram and then enter the meter group details relating to that premises. In Fig.1 you will see that the first premises is called office block and it has 2 meter groups, Floor 1 & Floor 2.
When the meter groups are created you can then add each individual meter using the "New Meter Details" dialog box. This will allow you to assign each meter to each of the meter groups. In the above example floor 2 has 2 meters: Canteen B and Server Room. Once all meters, meter groups and premises are added your configuration is complete.

*On initial setup of the system an e-MissionManagement technical assistant will help you configure your organisational structure.
