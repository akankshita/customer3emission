Conversion Factors

This section can be found by clicking Admin > Conversion Factors from the main menu.

This section allows the users to view the conversion factors for both electricity and gas usage. It also allows the users to set custom conversion factors depending on their organisations utilities provider.

The conversion factors page includes two tables, listing electricity conversion factors and gas conversion factors. Fig.42 and Fig. 43 depicts these tables respectively.

Fig.42 Electricity Conversion Factors

The table above lists 3 columns of information.

  • Conversion Factor: This is the actual value that has been applied to all electricity usage.
  • Year: The year this conversion factor was used for.
  • Official or Custom Value: This displays whether an official value was used or a custom value was used.
Fig.43 Gas Conversion Factors
  • Conversion Factor: This is the actual value that has been applied to all gas usage.
  • Year: The year this conversion factor was used for.
  • Official or Custom Value: This displays whether an official value was used or a custom value was used.
