Entering a Custom Conversion Factor

If the user wishes to use a custom conversion factor (provided by a utility provider), the user must click the "New Conversion Factor" button. This is depicted below in Fig.44

Fig.44 Electricity Conversion Factors

Once the user has clicked the New Conversion Factor button they will be taken to a page to add the custom conversion factor. The page will look like Fig.45 below.

Fig.45 Add Conversion Factor

The form above allows the user to create a custom conversion factor. It contains 3 options.

  • Source Type: Whether the conversion factor relates to gas or electricity usage.
  • Year: The year it will apply to.
  • Conversion Factor: The value of the new conversion factor.

Once created the user will be brought to the conversion factor page and the custom conversion factor added will be highlighted in red. A message will appear informing the user that the new conversion factor was successfully created. Fig.46 below depicts this.

Fig.46 Custom Conversion Factor Added

Each custom factor has two buttons as per the above diagram.

  • Edit: This allows the user to edit the conversion factor
  • Delete: Deletes the specified conversion factor from the e-MissionManager application.

*Note - For more information on conversion factors please contact the relevant regulatory body for the latest released conversion factors
