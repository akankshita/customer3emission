Document Management System

This section covers the document management system within the e-MissionManager application. It acts as a central repository for user to store any documents related to their organisations energy usage. This includes, utility bills, invoices, receipts or any other type of documentation.

To upload a document, click on the Document Management in the main menu This should bring you to the document upload page as in Fig.22

Fig.22 Document Management System

To upload a document click on the Upload Document button

This will bring you to the document uploads form as in Fig.23

Fig.23 Document Upload Form
  • Name: Name of the document you are uploading.
  • Document Type: Select the type of document you are uploading (electricity, electricity bill or other).
  • Start Date: Choose a start date from the date picker that refers to the start date of the document. If it is a bill, the start date will be printed as the billing period.
  • End Date: Choose an end date from the date picker that refers to the end date of the document. If it is a bill, the end date will be printed as the billing period.
  • Purchase Order Number: This is optional, however most invoices/bills have a unique reference number. This can prove to be extremely useful to keep track of uploaded documents.
  • Choose file:Browse to the location of your document to be uploaded.

Once you have filled everything out click on the Save button.

You will be brought back to a list of all documents which have been uploaded already. You should also see a message stating the successful upload of the document.

Fig.24 Document Upload Successful

If you wish to make changes to the details of the document you uploaded Click the Edit button on the bottom left of the page (as above)

View all Uploaded Documents

To view a list of all documents uploaded click on the Document Management button in the main menu. You should see the document that you just uploaded, as in Fig.25

Fig.25 View All Document Uploads

There are now 4 options available for every document uploaded

  • Download: Downloads the document to your computer.
  • Show: Shows all the details of the particular document.
  • Edit: Allows you to edit the details of the document.
  • Delete: Deletes the document from the document management system.
View Associated Documents

There is also a method of viewing documents related to particular time periods or periods of energy usage.

On the main status graph (as illustrated in the Graph Analysis section) there are several data points. When you hover over a particular data point, a tooltip will appear showing you a more accurate view of that data point. Fig.26 below depicts this

Fig.26 Tooltip on Data Points

If you then click on this data point, 2 options will appear which ask you whether you would like to see the documents uploaded that relate to this particular period of energy usage or whether you would like to see all documents uploaded.
