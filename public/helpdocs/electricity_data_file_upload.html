Upload a File with Electricity Data

To upload a file click on the menu: Upload > Upload Electricity File

Fig.16 Upload Electricity Data File

Next click the Upload Electricity Data File Button

Fig.17 Upload Electricity Data File Button

This will bring you to a new upload electricity file page

Fig.18 Uploading the Data File
  • Choose File: Browse to the location of your data file.
  • Select Meter: Select the meter that this file relates to if this list is empty you need to follow the configure your meters guide (see Section 1 - Configuring Meters).
  • Units: This is the units that your electricity usage is measured in. This should be stated on your utility bill or your smart meter documentation.
  • Interval (minutes between readings): Enter the number of minutes between timestamps in your upload file. If you have an hourly reading electricity meter than this value would be 60.
Once you have filed in these values than click the Create button.

You will be brought back to a list of all electricity data files which have been uploaded already. You should also see a message stating the successful upload of the electricity data file.
Fig.19 Electricity File Uploaded Successfully
View all Electricity Data

To View all previous electricity data file and electricity reading uploads click on the menu Upload > Upload electricity File

Then click on the View Electricity Readings Data button as show in Fig.20

Fig.20 View all Electricity Data
You should now see a table containing all the uploaded electricity data files as in Fig.21
Fig.21 View Electricity Readings Data

  • You can click Show to view the data for that reading.
  • You can click Edit to alter the details of an uploaded file.
  • You can click Delete to remove an upload file and its electricity readings.
