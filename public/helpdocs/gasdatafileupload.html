Upload a File with Gas Data

To upload a file click on the menu: Upload > Upload Gas File

Fig.6 Upload Gas Data File

Next click the Upload Gas Data File Button

Fig.7 Upload Gas Data File Button

This will bring you to a new upload gas file page

Fig.8 Uploading the Data File
  • Choose File: Browse to the location of your data file.
  • Select Meter: Select the meter that this file relates to. If this list is empty you need to follow the configure your meters guide (refer to the Configuring Meters).
  • Units: This is the units that your gas usage is measured in. This should be stated on your utility bill or your smart meter documentation.
  • Interval (minutes between readings): Enter the number of minutes between timestamps in your upload file. If you have an hourly reading gas meter than this value would be 60.
Once you have filed in these values than click the Create button.

You will be brought back to a list of all gas data files which have been uploaded already. You should also see a message stating the successful upload of the gas data file.
Fig.9 Gas File Uploaded Successfully
View all Gas Data

To View all previous gas data file and gas reading uploads click on the menu Upload > Upload Gas File

Then click on the View Gas Readings Data button as show in Fig.10

Fig.10 View Gas Readings Data

You should now see a table containing all the uploaded gas data files as in Fig.11
Fig.11 View Gas Readings Data

  • You can click Show to view the data for that reading.
  • You can click Edit to alter the details of an uploaded file.
  • You can click Delete to remove an upload file and its gas readings.
