Printing and Emailing the Graphs

All graphs throughout the e-MissionManager application can be printed and emailed. This feature is extremely useful if abnormalities or irregularities in energy usage are found so they can be addressed by members of the board, different departments, staff etc.

In the top-right of each graph you will see 2 buttons. The left-most button displays the image of a printer and allows the user to print the graph and its contents. The right-most image allows the user to download the image in a variety of formats, so they can be attached to email.

Fig.40 below depicts these buttons

Fig.40 Download and Print buttons

The print button will automatically find your default printer for printing the document.

  • PNG: Portable Network Graphics - High quality and no loss of quality.
  • JPG: Most common format of image. Loss of quality however on resize.
  • PDF: Adobe's Portable Document Format - Document.
  • SVG: Scaleable Vector Image.
